# Анализ спроса на специалистов с ИИ-навыками в России

**Автор:** Юров Сергей  
**Дата:** Март 14, 2026  
**Стек:** Python (pandas, matplotlib, seaborn), Jupyter Notebook

## 📌 Описание проекта

Исследование рынка труда для специалистов с навыками в области искусственного интеллекта (ИИ) на основе данных о вакансиях за 2022–2024 годы.

**Цель:** Определить реальный спрос на ИИ-компетенции для помощи бизнесу в принятии кадровых решений.

## 🔍 Ключевые вопросы

- В каких секторах экономики наиболее активно ищут специалистов с ИИ-навыками?
- Каковы показатели спроса по федеральным округам и субъектам РФ?
- Какие компании являются лидерами по найму?
- Как менялся спрос во времени?

## 📊 Основные выводы

- **IT-сектор** — абсолютный лидер спроса (61% всех вакансий). Это подтверждает, что именно IT-компании являются главными драйверами внедрения ИИ.
- **Москва и Центральный федеральный округ** формируют более половины всех вакансий, что говорит о сильной централизации ИИ-компетенций.
- **Топ-работодатели:** МТС, СБЕР, Газпром Нефть, VK, Яндекс. Присутствие Газпром Нефти в топ-5 показывает, что традиционные отрасли также активно внедряют ИИ.
- **Динамика:** Пик спроса пришелся на 2023 год, затем спад в 2024-м. Для подтверждения тренда необходим сбор свежих данных.
